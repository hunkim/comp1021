Speeding Up the Turtle * The turtle initially moves fairly slowly * You can adjust the moving speed of the turtle using the turtle.speed command * The initial speed value is 3 * For example, you can make the turtle move faster by running this line of code: `turtle.speed(8)` # The Turtle Speed Value * The speed value is an integer from 0 to 10 * The speed changes from very slow to very fast from 1 to 10, i.e. * 1=slow speed … 6=normal speed … 10=fast speed * A speed value of 0 means no animation at all * If you use the value of 0 you will still see the turtle ‘jump’ immediately from one position to another instead of moving gradually #
