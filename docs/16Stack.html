Implement a stack using Python code ## What is a Stack? * A stack is a special kind of data structure * It is used to store a collection of things * Sometimes it is a called a Last In First Out (LIFO) structure, because of the way it works ![image](https://user-images.githubusercontent.com/901975/66490110-3c339500-eae3-11e9-8df5-309f8b12f71b.png) ## Stack Operations * There are two operations for a stack: * Push: It adds a new item to the top of the stack * Pop: It takes the top item off the stack ![image](https://user-images.githubusercontent.com/901975/66490276-7dc44000-eae3-11e9-9abe-36bfd22d6fcf.png) ## An Illustration of a Stack ![image](https://user-images.githubusercontent.com/901975/66490325-8f0d4c80-eae3-11e9-8dec-9e93db2bb757.png) ## Using the Stack in Python * Any Python list is a stack * There are two built-in functions for a Python list: * `list.append()` (this is the Python word for ‘push’) * `list.pop()` * The append operation is used to add a new item to the end of the list (= the top of the stack) * The pop operation is used to remove and return the last element in the list (= the top of the stack) # Create a list with two numbers in it numbers = [];
