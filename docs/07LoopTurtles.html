Understand repetitive rules ## While Loops in Turtle Graphics * Draw a square with a length of 10  * you need to ask the turtle to move forward and turn four times import turtle turtle.forward(100) turtle.right(90) turtle.forward(100) turtle.right(90) turtle.forward(100) turtle.right(90) turtle.forward(100) turtle.right(90) ## Drawing a Square Using a Loop * You can use a while loop to draw a square import turtle # Run the loop four times, # The loop will be executed # with these values, side = 0, 1, 2, 3 side = 0 while side < 4: turtle.forward(100) turtle.right(90) side = side + 1 ## Drawing a Star Shape * How about STAR? ![image](https://user-images.githubusercontent.com/901975/64471351-50ead900-d183-11e9-9254-95a460c0ac10.png) * What are the rules? import turtle side = 0 while side < 5: turtle.forward(100) turtle.right(144) side = side + 1 ## Generalization? import turtle turtle.speed(10) side = 0 side_num = 7 while side < side_num: turtle.forward(100) turtle.right(360*(side_num-3)/side_num) side = side + 1 ## Flower! Using Nested Loops * On the right is a flower image created by a single program * It’s a fun example of using loops, including nested loops * Stage 0: get the turtle graphics started * import the turtle module, open the window * Stage 1: create the curved stem * part of a circle * Stage 2: draw the petals * uses a nested loop * Stage 3: draw the central part * uses a loop ![image](https://user-images.githubusercontent.com/901975/64471594-2ea68a80-d186-11e9-8350-4647ad56234c.png) import turtle turtle.speed(0) turtle.width(10) turtle.color("green") turtle.up() # Don't draw while we move turtle.goto(100, -300) # Move the turtle to bottom right turtle.left(90) # Point the turtle upwards turtle.down() # Start drawing from now onwards turtle.circle(1000, 20) # Draw circle with radius 1000 ## Stage 2 – Drawing a Flower Using Nested Loops * We use a nested loop to draw the eight red petals of the flower, shown here:  * The inner while loop draws one petal using five circles * The outer while loop uses the inner while loop to draw eight red petals ![image](https://user-images.githubusercontent.com/901975/64471689-f6537c00-d186-11e9-8312-4579728077ba.png) import turtle turtle.color("red") petal_piece = 0 while petal_piece < 5: turtle.up() turtle.forward(petal_piece * 20) turtle.left(90) turtle.down() turtle.begin_fill() turtle.circle(15 + (petal_piece*5)) turtle.end_fill() turtle.up() turtle.right(90) turtle.backward(petal_piece * 20) turtle.down() petal_piece = petal_piece + 1 ## The Outer Loop – Drawing the Flower * The outer while loop uses the inner while loop 8 times, for 8 petals * Each petal has a different starting point, 50 pixels away from the center * The inner while loop draws five circles of a petal (this part is shown in the previous slide) * Here we go back to the starting point, the center * Each petal is drawn using a different angle by turning the turtle 45 degrees import turtle turtle.color("red") turtle.speed(10) petal = 0 while petal < 8: turtle.up() turtle.forward(50) turtle.down() petal_piece = 0 while petal_piece < 5: turtle.up() turtle.forward(petal_piece * 20) turtle.left(90) turtle.down() turtle.begin_fill() turtle.circle(15 + (petal_piece*5)) turtle.end_fill() turtle.up() turtle.right(90) turtle.backward(petal_piece * 20) turtle.down() petal_piece = petal_piece + 1 turtle.up() turtle.backward(50) turtle.down() turtle.left(45) petal = petal + 1 ## Stage 3 - Drawing the Flower Centre * The middle of the flower is drawn using a while loop, which draws a series of circles * We first draw the largest circle, then the smaller ones * We reduce the radius each time * If we reduced it by 3 pixels each time, we would get this. ![image](https://user-images.githubusercontent.com/901975/64471809-77f7d980-d188-11e9-8d92-ee6b7acd41a2.png) * However, let’s make a more interesting pattern like the bottom: * We do that by increasing the size of the reduction for each successive circle import turtle turtle.speed(10) turtle.color("yellow") turtle.pencolor("black") radius = 60 radius_reduction = 3 while radius > 10: turtle.up() turtle.forward(radius) turtle.left(90) turtle.down() turtle.begin_fill() turtle.circle(radius) turtle.end_fill() turtle.up() turtle.right(90) turtle.backward(radius) turtle.down() radius = radius - radius_reduction #radius_reduction = radius_reduction * 2 ## Hiding the Turtle * Sometimes the pictures in the turtle window will look better if the turtle is not visible inside the window * In our previous flower example, the turtle is in the middle of the flower when the drawing finishes * We hide the turtle near the end of the program using this line of code: `turtle.hideturtle()` * After running the code, the turtle is then hidden in the final result import turtle turtle.speed(10) turtle.color("yellow") turtle.pencolor("black") radius = 60 radius_reduction = 3 while radius > 10: turtle.up() turtle.forward(radius) turtle.left(90) turtle.down() turtle.begin_fill() turtle.circle(radius) turtle.end_fill() turtle.up() turtle.right(90) turtle.backward(radius) turtle.down() radius = radius - radius_reduction radius_reduction = radius_reduction * 2 turtle.hideturtle() ##
